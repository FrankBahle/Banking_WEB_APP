# Banking Application

The Banking Application is a web-based banking system developed using ASP.NET Web Forms, MySQL, and C#. The project simulates real-world banking operations by allowing users to manage customers, accounts, and transactions through a secure and user-friendly interface.

This project was built to apply backend development, database-driven application design, CRUD operations, and object-oriented programming principles in a practical banking system.

---

## Project Overview

The system allows users to perform core banking operations such as creating and managing customer profiles, managing bank accounts, recording deposits and withdrawals, and viewing account transaction history.

It demonstrates how a banking system can use a structured database and backend logic to manage financial records accurately and efficiently.

---

## Features

* Customer profile management
* Account creation and account updates
* Deposit and withdrawal processing
* Transaction history tracking
* CRUD operations for customer and account records
* Database-driven data storage
* User-friendly web interface
